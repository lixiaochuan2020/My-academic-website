# Xiaochuan Li - Academic Website

Personal academic website built with [Hugo](https://gohugo.io/) using the [academimal](https://github.com/lixiaochuan2020/academimal) theme.

## Getting Started

First, install [Hugo](https://gohugo.io/installation/).

Then run the development server:

```bash
hugo server
```

Open [http://localhost:1313](http://localhost:1313) with your browser to see the result.

## Deployment

The site is deployed via GitHub Pages using a GitHub Actions workflow.
